#### **1. Стек технологий 

**Frontend (VR-клиент):**

- **Движок:** Unity 2022 LTS (+ XR Interaction Toolkit) или Unreal Engine 5
    
- **Язык:** C# (для Unity) или C++/Blueprints (для Unreal)
    
- **Платформа:** PICO
    
- **Плагины:** Pico CMS для Nextcloud
    

**Backend (для релиза 2+):**

- **Бэкенд-сервис:** Firebase / Supabase / или простой Node.js на Heroku
    
- **База данных:** Firebase Firestore / PostgreSQL
    
- **Хранилище файлов:** Firebase Storage / AWS S3
    
- **Аутентификация:** Firebase Auth
    

**Инфраструктура:**

- **Система контроля версий:** Git + GitHub
    
- **CI/CD:** GitHub Actions (для автоматической сборки APK)
    
- **Дистрибуция:** Meta Quest App Lab
    
- **Управление проектом:** GitHub Projects / Trello

#### **2. Melstone**

**Milestone 1: MVP Core — «Рабочий прототип»**  
*Срок: 6-8 недель*

✓ Настройка Unity/Unreal с VR поддержкой
✓ Базовая VR навигация (перемещение, поворот)
✓ Система взаимодействия с объектами (взять/переместить)
✓ Библиотека из 15 базовых 3D-моделей мебели
✓ Простой UI в VR-пространстве
✓ Локальное сохранение сцены
✓ Стабильная работа на PICO (72 FPS)

**Milestone 2: Release 1 — «Базовый конструктор»**  
*Срок: 4-6 недель после MVP*

✓ Конструктор комнат (стены, окна, двери)
✓ Расширенная библиотека (50+ моделей)
✓ Система материалов (стены, полы)
✓ Улучшенный UI/UX
✓ Оптимизация производительности
✓ Публикация в PICO STORE
✓ Сбор базовой аналитики

**Milestone 3: Release 2 — «Конструктор для города»**  
*Срок: 8-10 недель после Release 1*

✓ Облачное сохранение проектов
✓ Система пользователей (регистрация)
✓ Интеграция с каталогами партнеров
✓ Калькулятор бюджета
✓ Экспорт спецификации
✓ 2D-просмотр на сайте
✓ Модуль для партнеров (добавление товаров)
