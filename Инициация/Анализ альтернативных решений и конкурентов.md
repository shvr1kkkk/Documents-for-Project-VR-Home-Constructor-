#### **1. Анализ альтернативных решений (Чем люди решают проблему сейчас?)**

Перед запуском нового продукта пользователи пользуются существующими, хотя и менее эффективными, методами.

|Решение|Описание|Недостатки (Почему это не идеально)|
|---|---|---|
|**2D-планировки на бумаге**|Рисование от руки или схемы в графических редакторах (Photoshop, простые редакторы).|Полное отсутствие ощущения пространства, масштаба и объема. Невозможно оценить эргономику.|
|**Простая мебель «на глаз»**|Покупка мебели без точного плана, примерка по габаритам в магазине.|Высокий риск ошибки: мебель не вписывается в пространство, не проходит в дверь, не сочетается по стилю.|
|**Фотошоп-коллажи**|Создание примерного изображения будущего интерьера из фотографий мебели.|Статичное, необъективное изображение. Невозможно оценить вид из разных углов комнаты, освещение.|
|**Бесплатные онлайн-планировщики** (Planner 5D, Roomstyler, Sweet Home 3D и др.)|**Основные конкуренты в сегменте DIY.** Позволяют создать 3D-модель квартиры.|**Ключевая слабость:** Отсутствие **иммерсивности**. Пользователь остается "богом", смотрящим на макет сверху, а не "жителем" пространства. Нет ощущения масштаба.|
|**Услуги профессионального дизайнера**|Дизайнер создает 3D-визуализации (часто в 3ds Max + Corona/V-Ray).|**Дорого и долго.** Цикл от идеи до визуализаций — 2-4 недели. Каждая правка занимает время. Клиент может быть недоволен результатом после всех ожиданий.|
|**Дорогие профессиональные ПО** (3ds Max, SketchUp + V-Ray, ArchiCAD)|Мощные инструменты для архитекторов и визуализаторов.|**Высокий порог входа.** Требуются месяцы обучения. Не предназначены для быстрого моделирования конечным пользователем.|

**Вывод по альтернативам:** Существующие решения представляют собой компромисс между затратами (деньги/время) и качеством результата. Рынок ощущает вакансию для инструмента, который дает **качество и точность профессионального решения с простотой и скоростью DIY-планировщика**, дополненные ключевым элементом — **эффектом присутствия**.


#### **2. Анализ прямых и косвенных конкурентов**

Анализ строится на сравнении по ключевым параметрам, важным для пользователя: иммерсивность, простота, стоимость и функционал.

**Прямые конкуренты**

|Проект / Компания|Описание|Преимущества|Недостатки (Наши возможности для дифференциации)|
|---|---|---|---|
|**IKEA Place (и аналоги)**|Мобильное AR-приложение для размещения единичных моделей мебели IKEA в комнате.|**Простота использования,** сильный бренд, точные 3D-модели.|**Очень ограниченный функционал:** только мебель IKEA, нельзя создать целый интерьер, нет работы со стенами, светом, отделкой. Нет VR.|
|**VR-приложения от премиальных застройщиков**|Например, "Самолет", "ПИК" и др. создают VR-туры по готовым квартирам.|**Высокое качество графики,** эффективно для демонстрации.|**Отсутствие кастомизации.** Это готовый, неизменяемый тур. Пользователь не может поменять отделку или переставить мебель. Цель — продать, а не дать инструмент для творчества.|
|**Professional VR-визуализация**|Студии дизайна, использующие VR-плагины для Unreal Engine/Twinmotion.|**Фотореализм,** полный контроль над сценой.|**Экстремально высокая стоимость** (тысячи долларов за проект), **долгий процесс,** требуется специалист. Не массовый продукт.|

**Косвенные конкуренты

|Проект / Компания|Описание|Преимущества|Недостатки (Наши возможности для дифференциации)|
|---|---|---|---|
|**Planner 5D**|Лидер среди онлайн-конструкторов. Есть режим "Виртуальный визит" (псевдо-VR на экране).|**Огромная база пользователей,** интуитивный интерфейс, кроссплатформенность (Web, iOS, Android).|**"Виртуальный визит" — это не истинное VR.** Это режим от первого лица на плоском экране, без полного погружения и ощущения масштаба. Нет интеграции с популярными VR-шлемами.|
|**Roomstyler (от Easyhome)**|Онлайн-планировщик с большим каталогом реальных брендов.|**Хороший каталог реальной мебели,** простой интерфейс.|**Отсутствие иммерсивных технологий,** только 3D-вид и 2D-планы. Устаревший по сравнению с современными движками визуал.|
|**Sweet Home 3D**|Бесплатное настольное ПО с открытым исходным кодом.|**Полностью бесплатен,** мощный для своего класса.|**Сложный для новичка,** визуал сильно уступает современным решениям, нет онлайн-сохранения и совместной работы.|
|**Houzz**|Платформа-аггрегатор идей и магазин. Есть инструмент "View in My Room" (AR).|**Огромная база вдохновения,** сильное комьюнити, интеграция с ритейлерами.|**Инструмент планирования вторичен.** AR-функционал ограничен, как у IKEA. Нет комплексного подхода к проектированию всего помещения.|
